# Board - VueJS

Simple board (like Kanban Board) with Vue.js

## Demo
[http://itsnunolemos.github.io/board-vuejs/app/](http://itsnunolemos.github.io/board-vuejs/app)


## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
gulp watch
```
